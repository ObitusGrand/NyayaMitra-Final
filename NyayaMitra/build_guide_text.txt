For NyayaMitra to be trustworthy and market-ready, every law section cited must come from official government sources — not third-party websites, not Wikipedia, not scraped data. Here is where to get every statute.

3.1 Primary Source — India Code (indiacode.nic.in)
India Code is the official Government of India repository of all central legislation. Every Act available here is the authoritative, legally binding version. It is free, always available, and provides downloadable text files.

URL: https://www.indiacode.nic.in
Format: PDF and HTML versions available for every Act
Authority: Ministry of Law and Justice, Government of India
Coverage: All central Acts from 1836 to present
How to download: Go to Act name → Click 'PDF' → Download → Convert to text using pdfplumber

3.2 Amendment Source — eGazette (egazette.gov.in)
URL: https://egazette.gov.in
Authority: Department of Publication, Ministry of Housing and Urban Affairs
What it has: Every official gazette notification — ALL amendments are published here first
How to use: Search by Act name, date range, or ministry — download PDF notifications
For NyayaMitra: Scrape weekly, look for keywords: amendment, substitution, insertion, repeal

3.3 Court Judgments — Indian Kanoon API
URL: https://api.indiankanoon.org
Coverage: 10+ million Indian court judgments from Supreme Court, High Courts, Tribunals
Cost: Free developer tier available — sufficient for ML training data
How to use for NyayaMitra: Download judgments by case type for ML outcome predictor training
Registration: indiankanoon.org/api — request developer key

3.4 DLSA Office Data — NALSA
URL: https://nalsa.gov.in — National Legal Services Authority
What it has: All District Legal Services Authority (DLSA) office addresses, contacts, jurisdictions
How to use: Manually compile state-wise DLSA data into a JSON file for the DLSA map feature
Alternative: Use Google Maps Places API (free tier) to locate 'district legal services authority near me'

3.5 How to Convert PDF Laws to Text for ChromaDB
# Install pdfplumber
pip install pdfplumber

# Convert downloaded PDF to text
import pdfplumber

def pdf_to_text(pdf_path: str, output_path: str):
    with pdfplumber.open(pdf_path) as pdf:
        text = ''
        for page in pdf.pages:
            extracted = page.extract_text()
            if extracted:
                text += extracted + '\n'
    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(text)
    print(f'Converted: {output_path} ({len(text)} chars)')

# Convert all downloaded Acts
acts = [
    ('downloads/bns_2023.pdf',          'data/bns_2023.txt'),
    ('downloads/bnss_2023.pdf',         'data/bnss_2023.txt'),
    ('downloads/consumer_2019.pdf',     'data/consumer_2019.txt'),
    # ... add all Acts
]
for src, dst in acts:
    pdf_to_text(src, dst)
